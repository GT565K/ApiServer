== README


Info

* Simple RAILS API with REST endpoints for Company resource

* No client wrapper for API yet

Start-up
* configure database.yml for your local db

* make sure to use UTF-8 (unicode) encoding

* bundle install

* rake db:migrate

* rake db:seed to populate db

* use the generated ApiKey with the CURL commands found in scripts/curl_commands.txt for testing
